# idoremifa.github.io

Source for [idoremifa.github.io](https://idoremifa.github.io) — a personal tech blog by Ido. Notes on software, things I'm learning, and what I'm building.

Built with [Astro](https://astro.build/) on top of the [astro-paper](https://github.com/satnaing/astro-paper) theme (MIT). Deployed to GitHub Pages on push to `main`.

## Local development

```bash
pnpm install   # one-time
pnpm dev       # http://localhost:4321
pnpm build     # production build to dist/
```
